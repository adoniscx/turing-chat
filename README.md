# TuringChat

秉着学习的目的，探索性的做一个聊天界面。

# Quick Start

### 1. Clone the project.
```
git clone https://github.com/adoniscx/turing-chat.git
```

### 2. Opent in Xcode.
if (Welcome window show) : Click 'Open a project or file'  
elif : On navigationBar, Click 'File' -> 'Open' -> Select *your project folder*

### 3. Run.
cmd + r
